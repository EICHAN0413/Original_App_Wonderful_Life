# README 

## 開発言語
- Ruby2.6.5
- Rails5.2.5  
  

## 就業Teamの技術
- AWS EC2
- devise
- Ajaxを使ったコメント機能
- フォロー機能
- お気に入り機能  
  
## カリキュラム外の技術  
- vue.js  


## アプリケーションの実行手順
$ git clone git@github.com:EICHAN0413/Original_App_Wonderful_Life.git  
$ cd Original_App_Wonderful_Life  
$ bundle install  
$ rails db:create && rails db:migrate  
$ rails s
